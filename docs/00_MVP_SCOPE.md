## 복지용구 급여 처리 방침 (MVP)

```
표시:  상품에 is_insurance 플래그 + 급여 한도 월 / 연간 한도(고정 상수)
계산:  사용자가 직접 "잔여 한도"를 입력 → 프론트에서 계산 표시
저장:  elder_profiles.benefit_input jsonb { annual_limit, used_amount, self_pay_rate }
2차:  국민건강보험공단 API 연동 endpoint 설계만 docs에 보관
```
