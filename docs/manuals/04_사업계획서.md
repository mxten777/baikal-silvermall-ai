## 1. 사업 개요

### 1.1 미션

> **"장기요양 수급자와 가족이 복지용구를 가장 쉽고 저렴하게 이용하도록"**

### 1.2 사업 개요 요약

| 항목 | 내용 |
|------|------|
| 서비스명 | 실버몰 (SilverMall) |
| 사업 유형 | B2C 이커머스 플랫폼 (복지용구 전문) |
| 법인 형태 | 주식회사 |
| 설립 연도 | 2026년 |
| 주요 고객 | 장기요양 1~5등급 및 인지지원등급 수급자·가족 |
| 수익 모델 | 판매·대여 마진 + 파트너사 수수료 + AI 프리미엄 |
| 초기 투자 규모 | 3억 원 |
| 3년 목표 매출 | 60억 원 |
| 5년 목표 매출 | 200억 원 |

### 1.3 핵심 가치 제안

```
고객이 얻는 가치
├── 급여 한도 내 최저 본인부담금 (국가 지원 85%)
├── 등급별 맞춤 상품 자동 추천 (AI)
├── 복잡한 급여 청구 절차 자동화
└── 전국 익일 배송·무료 설치

경쟁사 대비 차별점
├── 급여 특화 필터링 (경쟁사 없음)
├── 본인부담금 실시간 계산기
├── 시니어 최적화 UX (큰 글씨, 음성 안내)
└── 케어매니저 B2B 파트너십
```

## 2. 시장 분석

### 2.1 거시 환경 (PEST 분석)

**정치·제도 (Political)**
- 장기요양보험 급여 품목 지속 확대 (2024년 구매 연 160만원, 대여 월 한도 별도)
- 제5차 장기요양 기본계획(2023~2027): 재가급여 확충 방향
- 노인장기요양보험법 개정으로 전자청구 의무화 진행 중
- 복지용구 급여 품목 2024년 기준 17개 품목 → 지속 확대 예정

**경제 (Economic)**
- 2024년 장기요양보험 총 지출: 약 14조 원
- 복지용구 급여 지출: 약 4,800억 원 (전년 대비 11%↑)
- 수급자 1인당 평균 복지용구 지출: 연 78만원
- 의료·복지 분야 GDP 대비 비중 지속 증가

**사회 (Social)**
- 2025년 65세 이상 인구: 약 1,050만명 (전체의 20.3%, 초고령사회 진입)
- 2030년 장기요양 수급자 추정: 약 130만명 (현재 110만명)
- 디지털 소외층인 시니어의 온라인 쇼핑 증가 (코로나 이후 가속)
- 가족 돌봄 부담 증가 → 온라인 구매 대리 수요 증가

**기술 (Technological)**
- AI/ML 기반 개인화 추천 기술 고도화
- 모바일 결제 보편화 (60대 이상 스마트폰 보유율 90%↑)
- IoT 기반 복지용구 원격 모니터링 기술 발전
- 클라우드 SaaS로 플랫폼 구축 비용 절감

### 2.4 타깃 고객 세그먼트

**Primary: 장기요양 가족 보호자 (35~55세)**
- 부모님 수발을 담당하는 자녀
- 디지털 친화적, 비교 구매 선호
- 평균 구매 주기: 6개월~1년

**Secondary: 케어매니저·사회복지사 (25~50세)**
- 수급자 대행 구매·추천 역할
- B2B 파트너 타깃
- 반복 구매, 대량 주문 가능

**Tertiary: 시니어 직접 구매자 (60~80세)**
- 디지털 리터러시 낮음 → UX 최적화 필수
- 음성 안내, 큰 글씨 모드 제공

## 3. 문제와 기회

### 3.1 현재 고객이 겪는 Pain Points

```
복지용구 구매 현재 여정 (평균 소요 시간: 2~3일)
────────────────────────────────────────────────────
1. 건강보험공단 홈페이지에서 급여 품목 목록 확인 (복잡)
      ↓ (30~60분)
2. 수급자 등급별 한도 금액 계산 (표 해독 어려움)
      ↓ (1~2시간)
3. 지역 오프라인 복지용구 센터 방문 또는 전화 문의
      ↓ (이동 시간 포함 반나절)
4. 업체별 가격 비교 불가 (가격 정보 불투명)
      ↓
5. 급여 청구 서류 수작업 작성
      ↓
6. 공단 승인 대기 (3~5 영업일)
────────────────────────────────────────────────────
→ 구매 완료까지 평균 7일 소요
```

**실버몰이 제공하는 해결책:**
```
1. 실버몰 접속 → 어르신 등급 입력 (30초)
2. AI가 맞춤 상품 추천 + 본인부담금 즉시 계산 (1분)
3. 찜 목록 비교 후 결제 (1분)
4. 급여 청구 자동 진행 (실버몰이 대행)
────────────────────────────────────────────────────
→ 구매 완료까지 3분 / 배송 익일 도착
```

### 3.2 시장 기회

1. **급여 시장의 온라인 전환 초기 진입**: 오프라인 독점 → 온라인 전환 초기 단계
2. **경쟁사의 전문성 부재**: 종합몰은 급여 체계 모름, 전문몰은 기술 낙후
3. **규제 완화 흐름**: 전자 급여 청구 확대로 온라인 구매 장벽 감소
4. **AI 추천 차별화**: 복잡한 급여 체계를 AI가 단순화 → 진입 장벽 구축

## 6. 기술 전략

### 6.1 기술 스택

| 레이어 | 기술 | 선택 이유 |
|--------|------|---------|
| Frontend | Next.js 14 + TypeScript | SSR·ISR, 접근성, SEO |
| Backend | Supabase (PostgreSQL + Edge Functions) | 실시간 DB, RLS 보안, 빠른 개발 |

| 결제 | Toss Payments | 국내 최적화, 간편결제 |
| 배포 | Vercel (CDN 글로벌) | 자동 CI/CD, Edge Network |
| 모니터링 | Vercel Analytics + Sentry | 실시간 성능·에러 모니터링 |

### 6.2 보안·규정 준수

- **개인정보보호법** 준수: 수급자 건강 정보 암호화 저장
- **의료기기법** 준수: 의료기기 해당 품목 인증 관리
- **장기요양보험법**: 급여 청구 전자 문서 표준 준수
- HTTPS 강제, RLS(행 수준 보안) 전 테이블 적용

- 고령층 소비 지출 증가
- 복지용구 시장 연평균 8~10% 성장 (2025~2030)

- 장기요양 인정자 수 2024년 기준 약 110만 명, 연 7% 증가
- 돌봄 가족(보호자)의 디지털 접근성 증가

- LLM 기반 개인화 추천 적용 용이

## 3. 문제와 기회

### 3.1 현재 시장의 문제

1. **복잡한 급여 처리**: 급여 대상 품목인지, 본인부담금이 얼마인지 소비자가 스스로 계산해야 함
2. **분산된 정보**: 급여 한도, 등급별 품목, 가격 정보가 공단 홈페이지·오프라인 센터에 분산
3. **구형 UX**: 기존 전문몰들은 모바일 최적화가 미흡하여 가족·보호자 이용 불편
4. **개인화 부재**: 어르신의 건강 상태·등급에 맞는 맞춤 상품 추천 없음
5. **대여 접근성 부족**: 단기·월 렌탈 서비스가 오프라인 중심으로 운영

### 3.2 실버몰의 솔루션

| 문제 | 솔루션 |
|------|--------|
| 복잡한 급여 계산 | 본인부담금 자동 계산 + 연간 한도 관리 도구 |
| 분산 정보 | 급여 가이드 + FAQ 콘텐츠 통합 제공 |
| 구형 UX | 모바일 퍼스트, 큰 글씨 지원, 탭바 | 
| 개인화 부재 | 어르신 프로필 기반 AI 추천 엔진 |
| 대여 접근성 | 온라인 월 렌탈 신청·관리 통합 |
