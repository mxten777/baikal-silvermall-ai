# 실버몰 사용자 매뉴얼

> 버전: 1.0.0 | 작성일: 2026-03-02  
> 대상: 실버몰 이용 고객 (장기요양 수급자·가족·보호자)

---

## 1. 실버몰이란?

**실버몰**은 장기요양 수급자를 위한 **복지용구 전문 쇼핑몰**입니다.  
국가 급여 품목을 판매·대여하며, **연간 160만 원 한도**의 국가 지원을 받아 본인은 **15%만 부담**하면 됩니다.

- 🛒 국가 급여 품목 구매·대여
- 💰 본인부담금 자동 계산
- 📦 빠른 배송 + 설치 서비스

---

## 2. 회원가입

### 2.1 가입 방법

1. 홈페이지(`silermall.co.kr`)에서 상단 우측 **사람 아이콘** 또는 **회원가입** 클릭
2. 이메일 주소와 비밀번호 입력
   - 비밀번호: 8자 이상, 영문+숫자 조합
3. **가입하기** 클릭
4. 입력한 이메일로 인증 메일 발송됨 → 메일 내 **이메일 인증하기** 클릭
5. 인증 완료 → 자동 로그인

### 2.2 로그인

1. 상단 우측 사람 아이콘 또는 하단 탭 **내정보** 클릭
2. 이메일과 비밀번호 입력 → **로그인**

### 2.3 비밀번호 찾기

1. 로그인 화면에서 **비밀번호를 잊으셨나요?** 클릭
2. 가입 이메일 입력 → 재설정 링크 발송
3. 메일의 링크 클릭 → 새 비밀번호 설정

## 3. 상품 찾기

### 3.1 카테고리 탐색

메인 화면의 카테고리 버튼에서 원하는 종류 선택:

| 카테고리 | 주요 상품 |
|---------|-----------|
| 🦽 이동편의용품 | 전동·수동 휠체어, 이동식 변기 |
| 🛏️ 욕창예방용품 | 공기매트리스, 욕창방지방석 |
| 🛁 세면·목욕용품 | 목욕의자, 샤워카트 |
| 🚽 배변처리용품 | 이동식 변기, 변기 보조기구 |
| 👁️ 감각·인지기능 | 시각·청각 보조기구 |
| 🧠 치매·인지증 | 인지훈련 도구 |
| 💨 호흡기·흡인기 | 흡인기, 산소발생기 |

### 3.2 검색

1. 상단 돋보기 아이콘 클릭 또는 검색창 터치
2. 상품명 또는 키워드 입력 후 **검색** (`예: 휠체어`, `목욕의자`)
3. 검색 결과에서 원하는 상품 선택

### 3.3 상품 필터·정렬

스토어 화면 우측 상단 **필터(≡)** 버튼:

| 옵션 | 설명 |
|------|------|
| 전체 / 구매 / 대여 | 상품 유형 필터 |
| 급여 품목만 | 국가 지원 대상 상품만 표시 |
| 인기순 | 많이 팔린 상품 우선 |
| 신규순 | 최근 입고 상품 우선 |
| 낮은 가격순 / 높은 가격순 | 가격 정렬 |

## 4. 상품 상세 확인

### 4.1 확인 항목

- **상품 이미지**: 좌우 스와이프로 여러 이미지 확인
- **급여 배지**: 🟢 `급여` 표시 = 국가 지원 대상 품목
- **가격 정보**:
  - 정가: 회색 취소선 가격
  - 판매가: 굵은 청록색 가격
  - **급여 본인부담**: `급여 본인부담 ○○○원 ~` (15% 기준 예상 금액)
- **대여 가능**: 🔵 `대여가능` 배지 표시 시 월 렌탈 방식으로 이용 가능

### 4.2 옵션 선택

일부 상품은 사이즈·색상 등 옵션이 있습니다:
1. 원하는 옵션 클릭
2. 재고 확인 (품절 옵션은 선택 불가)
3. 수량 조절 (＋/－ 버튼)

## 5. 장바구니

### 5.1 상품 담기

- 상품 카드 위에 마우스를 올리면 🛒 아이콘 표시 → 클릭 시 바로 담기
- 또는 상품 상세 → **장바구니 담기** 버튼

### 5.2 장바구니 확인

1. 상단 장바구니 아이콘 또는 하단 **내정보** → 장바구니
2. 확인 가능한 정보:
   - 상품별 수량 변경 (＋/－)
   - 상품 삭제
   - 주문 금액 합계
   - 예상 급여 본인부담 합계

> 💡 로그인하지 않아도 장바구니에 담을 수 있으나, 로그인 후 주문 진행이 필요합니다.

## 6. 주문하기

### 6.1 주문서 작성

1. 장바구니에서 **주문하기** 버튼 클릭
2. **배송지 선택**:
   - 기존 배송지 선택 또는 **새 배송지 추가**
   - 받는 분 이름, 연락처, 주소 입력
3. **배송 요청사항** 입력 (선택)
4. **쿠폰 적용**:
   - 보유 쿠폰 목록에서 선택
5. **적립금 사용**:
   - 보유 적립금 전액 또는 원하는 금액 직접 입력
6. 최종 결제 금액 확인

### 6.2 결제

1. 결제 방법 선택: **카드 결제** / **가상계좌**
2. **결제하기** 클릭 → Toss Payments 결제창 표시
3. 카드 정보 입력 또는 가상계좌 발급 진행
4. 결제 완료 → 주문 완료 페이지 이동

> ⚠️ 결제 중 창을 닫으면 결제가 취소됩니다. 주문이 생성된 경우 마이페이지 → 주문 내역에서 확인 가능합니다.

## 7. 주문 조회 및 배송 확인

### 7.1 주문 내역 확인

1. 하단 탭 **내정보** → **주문 내역**
2. 전체 주문 목록 조회 (최신순)

### 7.2 주문 상태 안내

| 상태 | 의미 |
|------|------|
| 결제대기 | 가상계좌 입금 대기 중 |
| 결제완료 | 결제 확인 완료 |
| 상품준비중 | 상품 포장·출고 준비 |
| 배송중 | 택배 기사 픽업 완료 |
| 배송완료 | 배송 완료 |
| 취소됨 | 주문 취소 처리 완료 |

### 7.3 배송 추적

1. 주문 상세 페이지에서 **배송 추적** 버튼
2. 택배사 페이지에서 실시간 배송 위치 확인

## 8. 교환·반품 신청

### 8.1 신청 가능 기간

- **교환·반품**: 배송 완료일로부터 **7일 이내**
- 상품 하자·오배송의 경우 **30일 이내**

### 8.2 신청 방법

1. 마이페이지 → 주문 내역 → 해당 주문 클릭
2. **교환 신청** 또는 **반품 신청** 버튼 클릭
3. 사유 선택 및 상세 내용 입력
4. **신청** → 담당자 검토 후 수거 안내 연락

### 8.3 교환·반품 불가 항목

- 사용·설치 후 정상 상품
- 포장 파손·분실
- 상품 손상 (고객 귀책)
- 급여 처리된 상품 (공단 환수 문제)

---

## 9. 리뷰 작성

1. 마이페이지 → 주문 내역 → 배송 완료된 상품에 **리뷰 쓰기**
2. 별점(1~5) 선택
3. 솔직한 사용 후기 작성 (최소 10자 이상)
4. 사진 첨부 (선택, 최대 5장)
5. **등록** → 적립금 100원 자동 지급

---

## 10. 상품 문의

1. 상품 상세 → **Q&A** 탭 → **문의 작성**
2. 문의 종류 선택: 배송·가격·사이즈·기타
3. 비밀 문의 여부 선택
4. 내용 입력 후 **등록**
5. 답변 완료 시 문자·앱 알림 발송

> ⏱️ 답변 소요 시간: 평일 기준 1~2 영업일 이내

## 12. 어르신 프로필 등록 (급여 추천 활용)

1. 마이페이지 → **어르신 프로필** → **프로필 추가**
2. 입력 항목:
   - 별명 (예: "어머니", "할아버지")
   - 생년월일·성별
   - 장기요양 등급 (1~5등급 / 인지지원등급)
   - 간단한 건강 상태 설문 (이동·목욕·배변 불편 여부)
3. 저장 후 스토어에서 **해당 프로필 기반 추천 상품** 자동 표시

## 13. 급여 한도 안내

### 국가 지원 급여 혜택

| 구분 | 내용 |
|------|------|
| 대상 | 장기요양 인정서 보유자 (1~5등급, 인지지원등급) |
| 연간 한도 | 구매 160만원 / 대여 연간 산정 |
| 본인부담 | 일반: 15% / 감경: 6~9% / 의료급여: 면제 |

### 본인부담금 계산 예시

```
전동침대 정가 800,000원
→ 급여 본인부담 = 800,000 × 15% = 120,000원
→ 국가 지원 = 680,000원
```

> 📋 실제 지원 금액은 공단 인정 기준 및 잔여 한도에 따라 다를 수 있습니다.  
> 정확한 한도 확인: **국민건강보험공단 노인장기요양보험** (www.longtermcare.or.kr)

## 14. 자주 묻는 질문 (FAQ)

**Q. 회원가입 없이 주문할 수 있나요?**  
A. 장바구니에 담기는 가능하나, 주문·결제는 회원 로그인이 필요합니다.

**Q. 급여 품목은 어떻게 구분하나요?**  
A. 상품 카드 및 상세에 초록색 `급여` 배지가 표시됩니다.

**Q. 대여 상품은 어떻게 반납하나요?**  
A. 대여 계약 만료 시 수거 일정을 안내해 드립니다. 마이페이지 → 주문 내역 → 해당 대여 상품에서 반납 신청 가능합니다.

**Q. 포인트(적립금)는 어떻게 사용하나요?**  
A. 주문서 작성 시 "적립금 사용" 항목에서 전액 또는 원하는 금액 입력 후 사용 가능합니다. (최소 1,000원 이상)

**Q. 결제 완료 후 주소 변경이 가능한가요?**  
A. 배송 준비 전(`preparing` 상태 이전)까지만 가능합니다. 고객센터 또는 1:1 문의로 연락해주세요.

**Q. 세금계산서 발급이 가능한가요?**  
A. 네, 사업자 고객의 경우 주문 시 사업자번호를 입력하시거나 고객센터로 문의해 주세요.
