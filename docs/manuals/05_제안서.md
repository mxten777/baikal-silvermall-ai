## 03. 솔루션 — 실버몰의 핵심 기능

### 3-1. 급여 최적화 엔진

수급자 등급(1~5등급/인지지원)을 입력하면, 남은 연간 급여 한도를 자동 계산하고 **본인부담금이 가장 적은 조합**을 추천합니다.

```
예시: 3등급 수급자, 연간 잔여 한도 80만원

추천 패키지 A (본인부담 12만원)
 ├── 전동침대  대여  급여 45만원 → 본인 6.75만원/월
 └── 이동식 좌변기 구매  급여 18만원 → 본인 2.7만원

추천 패키지 B (본인부담 8만원)
 ├── 에어매트리스 구매  급여 32만원 → 본인 4.8만원
 └── 샤워 의자 구매  급여 22만원 → 본인 3.3만원
```

### 3-2. AI 맞춤 추천

- 증상·생활환경·가족 구성 기반 자연어 입력
- 유사 수급자 구매 패턴 학습
- "이 분께는 이 조합이 좋습니다" 이유 설명 제공

### 3-3. 시니어 최적화 UX

| 기능 | 내용 |
|------|------|
| 큰 글씨 모드 | 글자 크기 160% 확대 |
| 간소화 모드 | 필수 정보만 표시, 단계 축소 |
| 전화 주문 연계 | "전화로 주문하기" 버튼 |
| 가족 대리 구매 | 수급자 계정 공유 기능 |

### 3-4. 케어매니저 B2B 포털


## 03. 솔루션: 실버몰 플랫폼

#### 🧮 급여 본인부담금 자동 계산
상품 카드와 상세 페이지에서 **정가 → 급여 본인부담금**을 즉시 표시합니다.

- 감경 대상자: 6~9% (수동 입력)
- 연간 잔여 한도 추적기 제공

1. 어르신 정보 등록 (등급, 이동 능력, 목욕 불편 여부 등)
2. → 맞춤 추천 상품 자동 표시

- 50~60대 보호자 중심 설계 (큰 글씨, 간단한 탐색)

- 대여 계약 기간, 반납 일정 앱에서 통합 관리
