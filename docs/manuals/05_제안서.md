## 01. Executive Summary

> "장기요양 110만 수급자가 복지용구를 가장 쉽고 저렴하게 이용할 수 있도록"

**실버몰**은 국내 유일의 **급여 특화 복지용구 전문 이커머스 플랫폼**입니다.  

고령화 사회에서 장기요양 수급자와 그 가족이 겪는 가장 큰 어려움은 **"어떤 상품을 얼마에 살 수 있는지"** 파악하는 것입니다. 실버몰은 이 복잡한 급여 체계를 단순화하고, AI 기반 개인 맞춤 추천을 통해 **본인부담금을 최소화**하는 스마트 쇼핑 경험을 제공합니다.

| 핵심 수치 | 내용 |
|-----------|------|
| 목표 시장 | 국내 복지용구 온라인 시장 (2027년 약 2,030억원) |
| 3년 목표 매출 | 60억원 |
| 핵심 기술 | Next.js + Supabase + AI 추천 엔진 |
| 투자 유치 목표 | 시드라운드 3억원 |

## 02. 우리가 해결하는 문제

```
장기요양 어르신 가족이 복지용구를 구매할 때 겪는 여정:
─────────────────────────────────────────────────────────
공단 홈페이지 → 급여 품목 목록 확인 (복잡)
      ↓
등급별 한도 계산 (표 해독)
      ↓
오프라인 복지용구 센터 방문 (또는 전화)
      ↓
가격 비교 불가 (업체별 가격 불투명)
      ↓
급여 청구 서류 작성
      ↓
→ 평균 소요 시간: 2~3일
```

**실버몰이 바꾸는 여정:**
```
실버몰 접속 → 어르신 등급 입력 → 맞춤 추천 확인
→ 본인부담금 즉시 확인 → 3분 내 주문 완료
```

## 03. 솔루션: 실버몰 플랫폼

### 3.1 핵심 기능

#### 🧮 급여 본인부담금 자동 계산
상품 카드와 상세 페이지에서 **정가 → 급여 본인부담금**을 즉시 표시합니다.
- 일반 수급자: 정가의 **15%**
- 감경 대상자: 6~9% (수동 입력)
- 연간 잔여 한도 추적기 제공

#### 🤖 어르신 프로필 기반 AI 추천
1. 어르신 정보 등록 (등급, 이동 능력, 목욕 불편 여부 등)
2. → 맞춤 추천 상품 자동 표시
3. → "어머니 등급에 맞는 전동침대 TOP 3"

#### 📱 모바일 퍼스트 UX
- 50~60대 보호자 중심 설계 (큰 글씨, 간단한 탐색)
- 하단 탭 네비게이션
- 터치 친화적 인터페이스

#### 🛒 원스톱 구매·대여
- 구매와 월 렌탈을 동시 지원
- 대여 계약 기간, 반납 일정 앱에서 통합 관리
- 재가 수급자 자택 배송·설치 서비스 연계

## 06. 기술 우위

### 6.1 현대적 기술 스택

```
사용자 브라우저
      ↕ (Next.js App Router - SSR/ISR/RSC)
Vercel Edge Network (CDN)
      ↕
Supabase (PostgreSQL 15 + RLS + Edge Functions)
      ↕
Toss Payments (결제 승인·웹훅)
```

- **Time to Market**: 기존 레거시 솔루션 대비 8배 빠른 개발
- **운영 비용**: 서버리스 아키텍처로 초기 서버 비용 **월 10만원 이하**
- **확장성**: 동시 접속자 10만 이상 대응 가능

### 6.2 IP / 기술 자산

| 자산 | 설명 |
|------|------|
| 급여 처리 엔진 | 등급·품목·한도 규칙 기반 계산 모듈 |
| 프로필 추천 알고리즘 | 건강 설문 → 상품 태그 매핑 엔진 |

## 07. 파트너십 제안

### 7.1 복지기관 대상 제안

| 제안 내용 | 기대 효과 |
|-----------|-----------|
| 기관 전용 계정 (무료) | 담당 어르신 복지용구 일괄 관리 |
| 직원 대량 구매 할인 (5~10%) | 수급자 가정 복지용구 접근성 향상 |
| 급여 가이드 콘텐츠 공유 | 기관 교육 자료로 활용 |
| 공동 홍보·캠페인 | 브랜드 신뢰도 상호 제고 |

### 7.2 제조사·공급업체 대상 제안

| 제안 내용 | 조건 |
|-----------|------|
| 독점 입점 (카테고리별) | 판매 수수료 15% |
| 공동 마케팅 | 콘텐츠 제작·광고비 공동 부담 |
| 데이터 공유 | 구매 패턴·VOC 분기 리포트 제공 |
| 재고 연동 API | 제조사 ERP → 실버몰 재고 자동 동기화 |

### 7.3 투자자 대상 제안

#### 시드 투자 조건

| 항목 | 내용 |
|------|------|
| 목표 유치 금액 | 3억원 |
| 투자 방식 | 지분투자 (SAFE 또는 구주) |
| 밸류에이션 | 논의 |
| 사용 계획 | IT 개발 30% · 재고 40% · 마케팅 20% · 운영 10% |

#### 투자 포인트

1. **빠른 성장하는 시장**: 장기요양 수급자 연 7% 증가, 온라인 전환 가속
2. **높은 진입장벽**: 급여 처리 복잡성·도메인 전문성이 자연적 해자
3. **반복 구매 구조**: 복지용구 소모품·대여 기간 연장으로 안정적 LTV
4. **정책 수혜**: 재가 케어 강화 정책으로 복지용구 수요 구조적 성장
5. **기술 완성도**: MVP 완성 + 실제 결제 테스트 가능 상태
